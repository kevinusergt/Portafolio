# Rock Band (HTML & CSS Assessment)

## Project Overview

This project is a **performance test** for a performance assessment focused on building a responsive landing page for a rock band.

---

# Technologies Used

* **HTML5**
* **CSS3**
* **JavaScript**
* **Google Fonts**


## How to Run the Project
1. Download the .zip file and extract it  
2. Open the project in your editor (e.g., VS Code)
3. Run it using Live Server or open:
```
index.html
```
